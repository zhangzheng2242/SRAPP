### 一、SRAPP项目是什么?

一个基于c++、QT整合了轻量化图像超分辨率深度学习模型的APP项目。可x2、x4放大图片分辨率（x2超分 输入400x400图片可以得到800x800图片），使低分辨率的图像更加清晰。

后续还可以整合其他深度学习模型如图像分类、目标检测等等。
